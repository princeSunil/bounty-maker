Import image as the wanted photo

Click IMPORT button to choose an image from your device storage, or drag and drop the image anywhere on this page.

Adjust the position of the photo

After importing the image, click(or touch) and drag the photo to adjust the position.

Resize the imported photo

- Move the cursor over the photo, and scroll the mouse wheel to zoom in or zoom out.

- Move the cursor over the red border around the photo and drag it to resize.

- For the touch screen device, touch two fingers on the top of the photo, and move them apart to zoom in, or together to zoom out.

Name and Bounty

Click EDIT button to open the edit panel, the first input field in the edit panel is name, and the second input field is bounty.

You can input any text for both of them, however if you input number in the bounty field, the Belly sign will be rendered in the front of the bounty text, and decimal comma will automatically insert in the text.

Install this app on your device

NOTE: For now, only Chrome and Edge support this feature.

This web application support PWA, which means you can install it as a native app on your device, and launch it directly without opening the browser.

Check out the installation guide.
